====================================
Xboinc: Xsuite BOINC Interface
====================================

Welcome to Xboinc's documentation!

Xboinc is a Python package that provides an interface for running Xsuite simulations on BOINC (Berkeley Open Infrastructure for Network Computing) distributed computing platforms.
